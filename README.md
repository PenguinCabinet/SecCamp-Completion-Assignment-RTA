# セキュリティキャンプ修了課題 RTA
[セキュリティキャンプ ジュニア/全国大会/ネクスト/コネクト](https://www.ipa.go.jp/jinzai/security-camp/index.html)の修了課題を終えるまでにかかった時間をランキングにしたものです。
## 注意
* あくまでネタリポジトリです。ここの順位が高いから偉いわけではありません
* 順位を狙うことより、クオリティを大切にしてください。結果的に早く提出で来てしまった人を、宝くじが当たったのように表彰するリポジトリです
## ルール
* IPAから修了課題の案内メールを受信したら計測スタート
* IPAから修了課題送信完了メールを受信したら計測終了
* ただし、ユーザー側の意図的な細工を除く不可抗力で修了課題送信完了メールを受信できない場合、ブラウザ上の提出完了画面表示をもって計測終了とし、ブラウザの閲覧履歴から計測終了時間を計算する
* フォーマットは`ユーザー名 - 計測時:計測分:計測秒`です。秒がわからない場合、00としてください。ユーザー名の部分をハイパーリンクにしたい場合、お好みでどうぞ
